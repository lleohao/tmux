# tmux
My tmux config

## Usage

### Install tpm 

> https://github.com/tmux-plugins/tpm

```
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```

### Download `tmux.conf`

```
wget https://raw.githubusercontent.com/lleohao/tmux/main/tmux.conf -O .tmux.conf
```

### Install plugins

Enter tmux session

```
$ tmux
```

Source config file

```
tmux source .tmux.conf
```

Install plugins
```
C-a I
```
