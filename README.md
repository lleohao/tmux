# tmux
My tmux config

## Usage

1. Install tpm 

> https://github.com/tmux-plugins/tpm

```
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```

2. Create `.tmux.conf` and copy `tmux.conf` to it

3. Install plugins

Enter tmux session

```
$ tmux
```

Source config file

```
tmux source .tmux.conf
```

Install plugins
```
C-a I
```
